# StockFlow Frontend UI

Modern, responsive dashboard built for the StockFlow Product Management System.

## Technologies Used
- **React.js (Vite)**: Lightning-fast frontend build tool and library.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI styling.
- **Axios**: Interceptor-based HTTP client for automatic JWT token injection.
- **React Router DOM**: Client-side routing for protected pages.
- **React Hot Toast**: Beautiful notification popups.

## Features
- **Dashboard Stats**: Real-time calculated inventory metrics (useMemo optimized).
- **Search & Filter**: Dynamic instant search and category filtering.
- **Role-Based UI**: Admins see "Add/Edit/Delete", Users see "Add to Cart".
- **Responsive Layout**: Works seamlessly on desktop and mobile.

## Getting Started
1. Run `npm install`
2. Run `npm run dev`
3. The app will launch on `http://localhost:5173`
